Delay discounting game

In these games, the player will make decisions in co-managing shared resources

  • Delay discounting game - money version

    • In this game, you will choose whether to receive less $ now or more with a delay

  • Delay discounting game - fish version

    • Same as the $ version except that instead of $, you will make decisions related to fishing

  • Delay discounting game - Forest version

    • Same as the $ version except that instead of $, you will make decisions related to a forest
