Public goods game

In these games, the player will make decisions in co-managing shared resources

  • Public goods game - gold version

    • In this game, the player will make decision to cooperate or deflect with another player in managing some shared golds

  • Public goods game - fish version

    • Same as the gold version except that instead of gold, you will co-manage fish rights

  • Public goods game - forest version

    • Same as the gold version except that instead of gold, you will co-manage a forest
